TweenMax学习笔记（1）

tweenMax是一款缓动插件，能很方便地实现很多动比较麻烦的动画效果，比如缓动动画，指定时间逐步进行的动画等，虽然一些动画CSS3也能实现，但明显要麻烦很多，所以花了一些时间来学习这款插件，如有不正确的地方，欢迎给与指正。

二、基本运动方法

1、终点式运动：t.to( 对象, 时间, { 属性：值 } ,延迟时间);

2、起点式运动：t.from( 对象, 时间, { 属性：值 } ,延迟时间);

3、有始有终运动：t.from( 对象, 时间, { 属性：值 } ,延迟时间);

三、多对象运动

1、t.staggerTo( 对象, 时间, { 属性：值 } , 延迟时间);

2、t.staggerTo( 对象, 时间, { 属性：值 } , 延迟时间);

.staggerFrom( )方法与.staggerTo( )方法的关系与 .to( )方法和.from( )方法的关系一样，所以.staggerFrom( )方法的使用方法与.staggerTo( )的使用基本一致，区别在于.staggerFrom( )方法是从定义在.staggerFrom( )方法里的位置运动到在初始的位置。。
