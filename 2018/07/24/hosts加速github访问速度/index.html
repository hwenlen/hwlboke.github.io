hosts加速github访问速度

大家都知道github网站是在国外的，所以在中国大陆访问的话有时候就会加载很慢，如何在不用翻墙软件的情况下加快github加载速度呢？下面谈谈我的解决方法。

一、找到hosts文件

不同系统hosts文件的位置也不一样
Windows 系统hosts位于 C:\Windows\System32\drivers\etc\hosts
Mac（苹果电脑）系统hosts位于 /etc/hosts

二、打开github搜索hosts

选择一个项目

打开项目中的hosts文件，Ctrl+F搜索github找到其中GitHub的内容，复制下来。

当然，有需要的话也可以全部复制下来

三、将复制下来的内容添加进自己电脑的hosts文件


保存，然后重启浏览器，你可以看到和原来比起来，现在github加载速度简直快得飞起。
